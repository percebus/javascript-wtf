A unit-test project to track all the weird JavaScript gotchas and test'em across all current browsers

## Setup

### 1st time

```bash
$> npm run setup
```

### BaU

```bash
$> npm install
```

## Test

### Jest

```bash
$> npm test
```

### Browsers

```bash
$> npm run testem:ci
```
